윈도우를 사용하면서 '이런 기능 하나쯤 더 있으면 좋겠다'고 생각한 적 없으신가요? 화면을 내 마음대로 나누고 싶거나, 웹사이트의 예쁜 색깔 코드를 바로 알고 싶을 때 말이죠. 이런 사소한 아쉬움을 해결해 줄 마법 같은 프로그램이 있습니다.

안녕하세요, 디지털 라이프의 질을 높여드리는 '일상감 라이프 스튜디오(ILSANGGAM)'입니다. 오늘은 여러분의 윈도우 PC에 슈퍼파워를 더해줄 마이크로소프트의 공식 무료 유틸리티, '파워토이(PowerToys)'를 소개해 드리겠습니다.

## 파워토이(PowerToys)란 무엇일까요?

파워토이는 마이크로소프트가 직접 개발하여 무료로 배포하는 윈도우용 고급 기능 모음집입니다. 원래는 과거 윈도우 버전에 있던 전문가용 도구였지만, 사용자들의 요청에 힘입어 최신 윈도우 환경에 맞게 새롭게 부활했습니다.

이 프로그램 하나만 설치하면, 윈도우에 기본적으로는 없던 강력하고 편리한 기능들을 마음껏 사용할 수 있게 됩니다. 복잡한 설정을 싫어하는 분들도 걱정 마세요. 필요한 기능만 골라서 켜고 끌 수 있어 매우 직관적입니다.

✅ **설치 방법**: 파워토이는 [Microsoft PowerToys 공식 GitHub 페이지](https://github.com/microsoft/PowerToys/releases) 또는 Microsoft Store에서 'PowerToys'를 검색하여 안전하게 다운로드할 수 있습니다.

## 생산성 200% 올려주는 필수 기능 TOP 3

파워토이에는 수많은 기능이 있지만, 그중에서도 설치 즉시 작업 효율을 극적으로 높여주는 핵심 기능 3가지를 엄선했습니다.


주변 지인에게도 추천했더니 만족도가 높았습니다.
### ① 창 정렬의 달인, FancyZones

여러 개의 창을 띄워놓고 작업할 때, 화면을 반으로 나누는 기본 기능만으로는 부족할 때가 많습니다. FancyZones는 사용자가 직접 화면 분할 레이아웃을 디자인하고, 단축키로 창을 지정된 구역에 착 달라붙게 만들어주는 기능입니다.

예를 들어, 모니터 왼쪽 절반에는 코딩 프로그램을, 오른쪽 위 1/4에는 참고 자료를 띄운 브라우저를, 아래 1/4에는 파일 탐색기를 배치하는 식의 커스텀 레이아웃을 만들어 사용할 수 있습니다. 드래그 한 번이면 창이 자석처럼 지정된 위치에 배치되어 멀티태스킹의 질이 달라집니다.

### ② 화면의 모든 색을 콕! Color Picker

디자인 작업을 하거나 PPT를 만들 때, 웹사이트나 이미지에서 본 마음에 드는 색상을 그대로 사용하고 싶을 때가 있습니다. 보통은 화면을 캡처해서 그림판이나 포토샵으로 색을 추출해야 했죠.

하지만 파워토이의 Color Picker를 사용하면 단축키 `Win + Shift + C`를 누르는 것만으로 화면 어디든 마우스 커서가 있는 곳의 색상 값을 바로 알 수 있습니다. HEX, RGB 등 다양한 형식으로 값을 복사할 수 있어 디자이너나 콘텐츠 제작자에게는 그야말로 필수 기능입니다.

### ③ 이미지 크기 변경을 1초 만에, Image Resizer

블로그에 올리거나 이메일로 보낼 사진 여러 장의 크기를 한 번에 줄여야 할 때, 일일이 그림판이나 포토샵을 여는 것은 매우 번거롭습니다. Image Resizer는 이 반복 작업을 혁신적으로 바꿔줍니다.

크기를 변경하고 싶은 이미지 파일들을 선택한 후 마우스 오른쪽 버튼을 클릭하고, '그림 크기 조정' 메뉴를 선택하기만 하면 끝입니다. 미리 설정된 크기(작게, 보통, 크게)를 선택하거나 원하는 사이즈를 직접 입력해 여러 장의 사진을 단 1초 만에 변환할 수 있습니다.

## 이 외에도 유용한 기능들

위에서 소개한 기능들 외에도 파워토이는 보석 같은 기능들을 품고 있습니다. 더 많은 테크 꿀팁은 [스마트 테크 가이드](/guide/smart-tech-guide)에서 확인해 보세요.

- **PowerToys Run**: `Alt + Space`로 실행하는 빠른 실행기. 프로그램, 파일, 계산까지 즉시 처리합니다.
- **Keyboard Manager**: 잘 쓰지 않는 키를 다른 키나 단축키로 바꿔주는 기능입니다.
- **File Locksmith**: "다른 프로그램에서 사용 중"이라며 삭제되지 않는 파일이 어떤 프로그램에 의해 잠겨 있는지 찾아줍니다.

## 자주 묻는 질문

**Q1: 파워토이는 윈도우 10에서도 사용할 수 있나요?**
A: 네, 파워토이는 윈도우 10 버전 2004 이상과 윈도우 11 모두에서 사용할 수 있습니다. 다만 일부 최신 기능은 윈도우 11에서만 지원되는 경우가 있으니 릴리스 노트를 확인해 보세요.

**Q2: 파워토이를 설치하면 컴퓨터가 느려지지 않나요?**
A: 파워토이는 매우 가볍게 설계되어 있으며, 필요한 기능만 개별적으로 켜고 끌 수 있어 시스템 리소스를 거의 차지하지 않습니다. 사용하지 않는 기능은 꺼두면 부담 없이 운영할 수 있습니다.

**Q3: 파워토이 업데이트는 어떻게 하나요?**
A: Microsoft Store에서 설치한 경우 자동 업데이트가 지원됩니다. GitHub에서 직접 다운로드한 경우에는 파워토이 설정 내 '일반' 탭에서 업데이트 확인 버튼을 눌러 최신 버전으로 업데이트할 수 있습니다.
