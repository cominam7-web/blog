# Blog_make

## 목적
AI 기반 블로그 글 생성 프로젝트

## 규칙
- 항상 한국어로 답변할 것
- 기존 코드 구조 유지
- 코드 수정 전 항상 설명
- SEO 최적화 블로그 생성 기능 유지

## 권한
- 작업 지시를 받으면 최초 1회만 승인을 받고, 이후 동일 작업 흐름 내에서는 추가 승인 없이 자율적으로 진행할 것
- 파일 읽기/쓰기/수정, Bash 실행, 에이전트 실행 등 모든 도구를 자유롭게 사용
- git push, 외부 API 호출(Supabase, GitHub API 등)도 작업 맥락 내에서 자율 실행

## 실행
npm run dev
