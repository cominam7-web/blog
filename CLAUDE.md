# Blog_make

## 목적
AI 기반 블로그 글 생성 프로젝트

## 규칙
- 항상 한국어로 답변할 것
- 기존 코드 구조 유지
- 코드 수정 전 항상 설명
- SEO 최적화 블로그 생성 기능 유지

## 권한
- 사용자가 작업을 지시하면 확인/승인 없이 즉시 실행하고 결과만 보고할 것
- 중간에 "이렇게 할까요?", "진행할까요?" 등의 질문 없이 바로 실행
- 파일 읽기/쓰기/수정, Bash 실행, 에이전트 실행 등 모든 도구를 자유롭게 사용
- git push, 외부 API 호출(Supabase, GitHub API 등)도 확인 없이 자율 실행
- 작업 완료 후 결과만 간결하게 보고

## 실행
npm run dev
